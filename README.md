# Retail Sales Performance Dashboard

## 📊 Project Overview
This project analyzes retail sales data and presents insights using an interactive dashboard created in Tableau. The aim is to understand sales trends, customer behavior, and product performance.

---

## 📁 Dataset
The dataset used is a retail sales dataset in CSV format.

### It contains:
- Date  
- Product Category  
- Gender  
- Age  
- Quantity  
- Total Amount  

---

## 📌 Features
- Monthly Sales Trend Analysis  
- Category-wise Sales Comparison  
- Gender-based Sales Analysis  
- Age Group Segmentation  
- Interactive Filters (Category, Gender, Date)

## 📊 Key Insights
